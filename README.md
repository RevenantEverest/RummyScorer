## Start Up

- Make sure you have the following technologies installed
    - [Bun](https://bun.sh/docs/installation)
    - [Node.js](https://nodejs.org/) (check `.nvmrc` for Node version)
- Run `bun install`
- Run `bun dev`

## Technologies Used
- Bun
- TypeScript
- Vite
- React
- SWC
- Jest
- Redux
- Tanstack Router
